Seletores CSS

Bem-Vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por tipo: Busca elementos por uma tag HTML específica
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor por Classe (.): Busca elementos através do atributo "class"
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo([atrib]): Seleciona elementos que possuem um atributo específico no documento HTML e conseguimos também buscar atributos com um valor específico

Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores com uma virgula(,).

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, podemos fazer um só bloco de CSS e reutilizar a mesma regra.
